Careers Job is plugin for collecting the live job vacancies from particular institution and shows on careers page of site.

== Description ==

Careers Job is plugin for collecting the live job vacancies from particular institution and shows on careers page of site. Use the  shortcode [careersjob_list] to display the list of job. First admin must have to enter/configure the institution id in admin panel of careers job plugin. And from there admin can customise the color of job listing table as per theme design / color.

Major features in Careersjob include:

* Automatically collect the all live job vacancies and filter out after enter the Institution ID.

* All filter data will display careers page of site using shortcode [careersjob_list].

* Admin can customise the job listing table color as per theme design/color.

PS: You'll must have to enter [InstitutionID] in admin panel of plugin, and use shortcode [careersjob_list] to display list of job.

== Installation ==

Upload the Careersjob plugin to your plugin directory, Activate it, then enter your [InstitutionID] in admin panel of plugin and use shortcode [careersjob_list] to display list of job.

1, 2, 3: You're done!

== Frequently Asked Questions ==

= For this plugin instituiton id should be exists edubreaks site ? =

Yes, institution id must be exists in edubreaks site/database. otherwise it will show the others instituiton id data.
